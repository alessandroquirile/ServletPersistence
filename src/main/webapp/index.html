Questa demo dimostra la persistenza di una servlet. Le variabili d'istanza della servlet permangono nel container finché non avviene il destroy del servlet (cioè quando viene fatto il reboot o si spegne il server contenente la servlet). Motivo per cui è possibile stabilire il numero di visitatori da quando la servlet è stata avviata per la prima volta (e finché non verrà spenta o riavviata).
